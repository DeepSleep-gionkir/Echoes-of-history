맞습니다. '파벌', '기업', '종교' 정도로는 현실의 그 복잡하고 미묘한 인간 군상을 다 담기에 부족하다고 느끼실 수 있습니다.

단순히 이름만 바꾸는 게 아니라, **[작동 메커니즘이 완전히 다른]** 상위 카테고리(Type)를 **8가지**로 확장해 봅시다. 이 정도면 1인 개발로 관리 가능한 범위 내에서 거의 모든 사회 현상을 시뮬레이션할 수 있습니다.

이것들을 **만능 엔티티 시스템**에 그대로 적용하면 됩니다.

---

### **확장된 8대 엔티티 유형 (The 8 Entity Archetypes)**

DB의 `type` 컬럼에 들어갈 분류입니다. 각 타입은 **플레이어에게 요구하는 것**과 **주는 효과**가 완전히 다릅니다.

#### **1. 파벌 (Faction) - "정치적 지분"**

- **기존:** 귀족, 상인 등.
- **메커니즘:** `충성도`가 낮으면 **정책(법안)을 거부**하거나 **쿠데타**를 일으킴.
- **핵심:** 국정 운영의 **찬성/반대**를 결정하는 세력.

#### **2. 기업 (Corporation) - "경제적 이익"**

- **기존:** 동인도 회사, 군수 공장.
- **메커니즘:** 매 턴 **골드**를 생산하지만, 특정 자원(철, 석탄)을 소비함. `이익`이 안 나면 **타국으로 이전(먹튀)**하거나 **로비(뇌물)**를 시도함.
- **핵심:** **돈**을 벌어다 주지만 **부패**를 유발하는 세력.

#### **3. 교단 (Cult/Religion) - "정신적 지배"**

- **기존:** 국교, 이단.
- **메커니즘:** 국경을 무시하고 타일 단위로 전파됨. `신앙심`을 자원으로 제공하지만, 교리에 어긋나는 행동(예: 과학 연구)을 하면 **민심(안정도)**을 깎음.
- **핵심:** **통제 불가능**하게 퍼지며 **이념**을 강요하는 세력.

#### **4. 신디케이트 (Syndicate) - "지하 경제" (NEW)**

- **정의:** 마피아, 밀수업자, 해적, 야쿠자.
- **메커니즘:**
  - 플레이어의 **골드를 훔쳐감(횡령/누수).**
  - 대신, **[더러운 일]**을 수행해 줌. (예: 암살, 타국에 마약 유통, 스파이 활동).
  - 경찰력을 동원해 **토벌**할지, 뇌물을 주고 **공생**할지 선택해야 함.
- **핵심:** **리스크(Risk)**를 관리하며 **불법적 이득**을 취하는 세력.

#### **5. 비밀 결사 (Secret Society) - "숨겨진 의도" (NEW)**

- **정의:** 프리메이슨, 혁명군, 일루미나티.
- **메커니즘:**
  - 평소에는 **보이지 않음(Hidden).** (충성도나 규모가 '?'로 표시됨).
  - 특정 조건(예: 독재가 심해짐)이 되면 갑자기 **정체**를 드러내고 **체제 전복**을 시도하거나, 반대로 왕권을 비밀리에 수호함.
  - 유저는 첩보 자원(`Authority`)을 써서 이들의 정체를 **수사**해야 함.
- **핵심:** **정보전**을 요구하며 **게임의 룰**을 뒤흔드는 세력.

#### **6. 학술 기관 (Institution) - "지식과 진보" (NEW)**

- **정의:** 왕립 학술원, 마법사 길드, 대학, 싱크탱크.
- **메커니즘:**
  - **골드를 소비**하여 **지식(Science)**이나 **권위(Authority)**를 생산함.
  - 가끔 **[위험한 실험]**을 제안함. (성공 시 대박 기술, 실패 시 수도 폭발).
  - 검열을 싫어하며, 자유를 억압하면 타국으로 망명함(두뇌 유출).
- **핵심:** **유지비**를 먹지만 **기술 발전**을 책임지는 세력.

#### **7. 지방 정부 (Local Gov) - "분리주의" (NEW)**

- **정의:** 식민지 총독부, 자치 도시, 소수민족 위원회.
- **메커니즘:**
  - 특정 **지역(타일 그룹)**을 기반으로 함.
  - 중앙 정부의 말을 잘 안 듣고 세금을 떼어먹으려 함.
  - 충성도가 바닥나면 **독립 선언**을 하고 새로운 국가(AI)로 떨어져 나감.
- **핵심:** **영토 관리**와 **분열**을 담당하는 세력.

#### **8. 길드/조합 (Union) - "노동과 파업" (NEW)**

- **정의:** 광부 조합, 운송 길드, 용병 조합.
- **메커니즘:**
  - 특정 산업(광산, 농장)의 **생산 효율**에 관여함.
  - 요구사항(임금 인상, 안전 보장)을 들어주지 않으면 **[파업]**을 일으켜 해당 산업을 마비시킴.
- **핵심:** **생산성**을 볼모로 잡고 **복지**를 요구하는 세력.

### **이것들이 "태그"와 만나면? (무한한 조합)**

이제 이 8가지 타입에 태그를 붙이면, 현실에 존재하는 거의 모든 조직을 만들 수 있습니다.

1.  **[신디케이트] + `Patriotic`(애국적) + `Maritime`(해양)**
    - = **사략 해적 (Privateers)**
    - 범죄 조직이지만 왕을 위해 적국 상선만 털어오는 "착한(?) 깡패".
2.  **[비밀 결사] + `Fanatic`(광신적) + `Violent`(폭력적)**
    - = **암살단 (Hashashin)**
    - 왕의 정적을 암살해주지만, 왕이 교리에 어긋나면 왕도 죽이려 듦.
3.  **[기업] + `State-Owned`(국영) + `Corrupt`(부패한)**
    - = **부패한 국영 기업**
    - 망하지는 않지만 세금을 블랙홀처럼 빨아먹음.
4.  **[지방 정부] + `Colonial`(식민지) + `Merchant`(상업)**
    - = **식민지 무역 거점**
    - 돈은 엄청 벌어다 주는데, 본국과 거리가 멀어서 통제가 안 됨.

### **구현 가이드 (1인 개발용)**

**"종류는 많아도 로직은 하나로 통일합니다."**

**DB Schema Update (`internal_entities`)**
`type` 컬럼에 위 8가지 문자열(`syndicate`, `secret_society` 등)이 들어갑니다.

**서버 로직 (Node.js)**
각 타입별로 **[매 턴 정산 함수]** 하나씩만 만들면 됩니다.

```javascript
// pseudocode
function processTurn(entity) {
  switch (entity.type) {
    case "corporation":
      // 돈 벌어오고, 환경 오염 시킴
      break;
    case "syndicate":
      // 돈 훔쳐가고, 범죄율 높임
      break;
    case "secret_society":
      // 음모 꾸미기 (랜덤 이벤트 트리거)
      break;
    // ... 나머지 타입 처리
  }

  // 공통: 태그에 따른 추가 보정
  if (entity.tags.includes("greedy")) {
    entity.loyalty -= 5; // 탐욕스러운 놈들은 매턴 충성도 하락
  }
}
```

이렇게 **8대 타입**으로 확장하면, "단순히 파벌만 있다"는 느낌은 사라지고 **"범죄 조직, 비밀 결사, 식민지 정부가 얽히고설킨 복잡한 제국"**을 표현할 수 있습니다.
